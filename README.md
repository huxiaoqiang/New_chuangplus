# chuangplus
##框架结构
后台框架：django+mysql
前台框架：angularjs
##环境配置
###python2.7
ubuntu 预安装  windows 自行百度
###pip
```
apt-get install python-pip
```
###安装所有 python 依赖包
```
pip install -r requirement.txt
```
###更新 python 依赖包列表
```
pip freeze > requirement.txt
```
###mysql
```
apt-get install mysql-server
```
###mysqldb module
```
apt-get install python-mysqldb
```
