# chuangplus
##框架结构
后台框架：django+mongodb
前台框架：angularjs
##环境配置
###python2.7
ubuntu 预安装  windows 自行百度
###pip
```
apt-get install python-pip
```
windows下，下载脚本运行
###安装所有 python 依赖包
```
pip install -r requirement.txt
```
###更新 python 依赖包列表
```
pip freeze > requirement.txt
```
###mongodb
数据库安装
###mongodbengine module
```
pip install mongoengine==0.8.7
```
